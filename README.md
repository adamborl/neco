# Jednoduchy Web

Tento repozitar obsahuje jednoduchou ukazku plnohodnotneho webu psaneho v HTML, CSS a JavaScriptu.

## Soucasti
- `index.html` – hlavni stranka s navigaci
- `style.css` – zakladni styly
- `script.js` – jednoduchy skript pro formular

Otevrenim souboru `index.html` v prohlizeci si muzete web prohlednout.
